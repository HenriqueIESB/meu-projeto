# Meu Projeto Git

Projeto simples para demonstrar o uso básico do
Git.

## Arquivos

- README.md

- index.html

- style.css

- script.js

### Aprendizado

Uso de Git: Durante o desenvolvimento, você irá versionar o
código, criando commits que refletem cada etapa do projeto

Fluxos de Trabalho: Durante o versionamento de código, você
aprenderá diferentes estratégias de fluxo de trabalho, e que o
GitHub Flow é uma estratégia aderente a práticas DevOps.
